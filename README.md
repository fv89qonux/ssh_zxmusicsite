## 本项目实现的最终作用是基于SSH实现简单在线听音乐收藏管理系统
## 分为2个角色
### 第1个角色为管理员角色，实现了如下功能：
 - 歌曲管理
 - 用户管理
 - 管理员登录
### 第2个角色为用户角色，实现了如下功能：
 - 按分类查看
 - 添加歌单
 - 用户登录
